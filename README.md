# PriceRadar BG

## 📌 Описание

PriceRadar BG е система за проследяване и анализ на цените на продукти в български онлайн магазини.

Целта на проекта е да осигури прозрачност на цените и да помогне на потребителите да разпознават реални намаления, резки поскъпвания и подвеждащи промоции.

---

## 🎯 Основни функции

* Проследяване на цени на продукти
* История на цените във времето
* Засичане на фалшиви намаления
* Откриване на резки промени в цените
* Сравнение на оферти между различни магазини

---

## 🧠 Как работи

Системата събира данни за цени, съхранява тяхната история и прилага логика за анализ, чрез която открива аномалии и дава полезна информация на потребителя.

---

## 🏗️ Архитектура

* Backend: FastAPI (`backend/`)
* Database: PostgreSQL (планирано)
* Frontend: React + TypeScript + Vite + Tailwind CSS (`frontend/`)
* Data collection: Scraping / API

### Frontend (UI основа)

```bash
cd frontend
npm install
npm run dev
```
